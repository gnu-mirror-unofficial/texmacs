GNU TeXmacs manual

For instructions on how to download and install TeXmacs, please consult the corresponding web pages:

  • Download and install TeXmacs.
  • Download and install TeXmacs using a Red Hat package.
  • Download and install TeXmacs using a Debian package.
  • Download and compile the source code.
  • Download extra fonts.

We recommend new TeXmacs users to browse through the TeXmacs tutorial on the web, which provides many examples and screenshots. Alternatively, you may download the entire tutorial, unpack it using 

    gunzip -c TeXmacs-tutorial.tar.gz | tar xvf -
and browse it without being connected to the web. Although the tutorial is still quite incomplete, it should address the most basic issues about TeXmacs. You are strongly invited to contribute to the tutorial or to translate it into other languages.

You may also download a more complete (but less pedagogic) TeXmacs manual in dvi format. If you have already installed TeXmacs, then you may also read this manual by typing 

    texmacs -x
Actually, this command is equivalent to 
    xdvi $TEXMACS_PATH/doc/dvi/texmacs &

Other documents which may be worthwhile reading:

  • GNU TeXmacs: A free, structured, wysiwyg and technical text editor, Actes du congrès GUTenberg (no. 39-40), Metz, mai 2001, pages 39-50.
  • TeXmacs interfaces to computer algebra systems by Andrey Grozin. Computer science e-print archive.
For those who want to port TeXmacs to other platforms, it may also be interesting to download additional miscellaneous test programs related to TeXmacs.
