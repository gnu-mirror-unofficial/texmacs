GNU TeXmacs manual

For instructions on how to download and install TeXmacs, please consult the corresponding web pages:

  • Download and install TeXmacs.
  • Download and install TeXmacs using a Red Hat package.
  • Download and install TeXmacs using a Debian package.
  • Download and compile the source code.
  • Download extra fonts.

Click here for the current TeXmacs manual in dvi format. You can also read this manual by typing 

    texmacs -x
after having installed TeXmacs. Actually, this command is equivalent to 
    xdvi $TEXMACS_PATH/doc/dvi/texmacs &

We have also started to write a more detailed and easier to read manual, which is available in postscript format or in TeXmacs format. After downloading the manual, you have to uncompress and untar it using 

    tar -zxvf TeXmacs-0.0.0.tar.gz
or 
    tar -zxvf TeXmacs-src.0.0.0.tar.gz
You are strongly invited to contribute to this manual. As soon as a TeXmacs to html converter will be available, you should also be able to browse this manual from the web pages.
