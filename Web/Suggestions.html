Suggestions made by users

Some not yet implemented suggestions by users of GNU TeXmacs

  • Two column format, floating objects, etc.
  • Plain TeX support.
  • Fold/unfold chapters, sections, sessions, etc.
  • A tag for the current date.
  • Inserting files as a complement to loading files.
  • Create python bindings (the scheme interface should first be more stable for this).
  • Gettext like interface for internationalization.


Other suggestions concerning the user interface

  • Interface to Gnome/Gtk/Guile. I am planning to replace the GUI by guile-gtk. Nevertheless, I will conserve my code for producing anti-aliased TeX fonts.
  • Several suggestions about window decorations. I will not spend to much time on this in the near future, until I have made up my mind about using guile-gtk.
  • Display progress bar when running Metafont.
  • When making a reference, show the list of active labels.
  • Dialogues, icons or key bindings for things like forced space, border around equation, and underline.
  • Have a configurable math toolbar menu where you can create template equations and save them.
  • Mouse/keyboard driven image and table resizing.
  • Implement the MS Word / Staroffice 'text move command' [mark text; left mouse click and pressed, move text to destination, release left mouse].
  • Optional automatic item creation in lists and enumerations each time when hitting return.
