Installation of GNU TeXmacs as a Red Hat Package


1. The idea behind rpm packages

If you use redhat linux or another linux distribution (like mandrake) which supports the rpm command to install software, it is very easy to install GNU TeXmacs. You can check whether rpm exists on your system using 

    which rpm
If the program is not installed, then you may download it from ftp://ftp.rpm.org/pub/rpm/dist/rpm-3.0.x and unpack it in the / directory as root. Notice that the rpm program can only be used by root. If you do not have root privilege, then you should ask your system administrator to install GNU TeXmacs, or use the classical installation method.

2. Download, install and run

After downloading the GNU TeXmacs distribution for PC's under GNU/Linux, you may install the software by typing (as root) 

    rpm -i TeXmacs-0.3.5.1-1.i386.rpm
The program can now be launched using 
    texmacs &
In a similar way, you may download the source code as an rpm package. You might also prefer to download the last stable version of TeXmacs, or the corresponding source code. We also provide rpm's for other operating systems.

3. Extra fonts?

Before you run TeXmacs, you may optionally download pregenerated TeX fonts and install them using 

    rpm -i TeXmacs-fonts-1.0-1.noarch.rpm
Although these fonts may in principle be generated automatically by TeXmacs, this process may slow down the editor when you use it for the first time.
