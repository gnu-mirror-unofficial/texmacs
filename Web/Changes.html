Changes

Changes from version 0.3.5.0 to 0.3.5.2

  • First implementation of mathematical input in computer algebra sessions (0.3.5.2).
  • Arbitrary predicates instead of modes and languages for keyboard mappings (0.3.5.2).
  • Variant of expand construct with inaccessible borders (0.3.5.2).
  • Started implementation of page insertions and multicolumn construct (0.3.5.2).
  • Computer algebra sessions are silently (re)started when feeding new input and the system is inactive (0.3.5.1).
  • Recursive kill on all subprocesses when closing a computer algebra session (0.3.5.1).
  • Fixed minor keyboard-related bugs (0.3.5.1).
